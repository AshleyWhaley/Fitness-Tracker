# Fitness-Tracker
 Simple application that logs workouts and displays them through a graphic dashboard. 

## Link
[FitnessTracker](https://thawing-lake-68040.herokuapp.com/)

## Installation
- npm i
- npm run seed
- npm start

## Technologies Used
MongoDB
Mongoose
Express
Logger
Node
